# E-Commerce Web Application

A responsive and modern e-commerce web application built using React.js.  
The project includes product listing, category filtering, dynamic routing, and shopping cart functionality.

---

## 🚀 Features

- Product Listing Page
- Category-wise Products (Men, Women, Kids)
- Dynamic Product Details Page
- Add to Cart Functionality
- Remove from Cart Functionality
- Cart Total Calculation
- Real-time Cart Item Counter
- Responsive User Interface
- React Router Navigation
- Global State Management using Context API

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- React Router DOM
- Context API
- HTML5
- CSS3
- Git & GitHub
- Vercel
